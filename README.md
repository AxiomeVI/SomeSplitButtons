# SomeSplitButtons

Inspired by [WonderMods](https://github.com/WonderGinger/WonderMods)

## Save and Quit Split
Add a menu button to unpause and split the room timer for practicing segments that end with S&Q

## Skip Cutscene Split
Add a menu button to unpause and split the room timer for practicing segments that end with a cutscene skip (end of chapter cutscenes on file time)

The `Number of cutscenes to ignore` option is used to ignore the first n cutscenes within the segment
