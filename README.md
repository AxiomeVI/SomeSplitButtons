# SomeSplitButtons

Inspired by [WonderMods](https://github.com/WonderGinger/WonderMods) Return to Menu split button

## Save and Quit Split
Add a menu button in the pause menu to unpause and split the room timer (after 31 frames) for practicing segments containing a S&Q. Optionally triggers a retry for death-warp strats, like the 3A heart room.

## Skip Cutscene Split
Add a button in the pause menu to unpause and split the room timer (after 18 frames or 232 for Prologue) for practicing end-of-chapter cutscene skips as if under filetime condition.
